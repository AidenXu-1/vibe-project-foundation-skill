# decisions —— 架构决策记录(ADR)

> ADR = Architecture Decision Record。每做一个**以后会被反复追问"当初为什么这么选"**的决定,就在这留一条。
> 一个决策一个文件,编号递增。复制 [`_template.md`](_template.md) 开始。

## 为什么要这个

开发到中期一定会问:「当初为什么选 X 不选 Y?」没记录就会反复纠结、甚至推翻重来。写下来,决策就有了"判例"。
